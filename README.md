# EDA-Titanic-Test-Dataset

🚀 Data Preprocessing & Feature Engineering Project | Jupyter Notebook

I recently worked on a dataset where I focused on **data cleaning, preprocessing, and feature engineering techniques** using Python in Jupyter Notebook. This project was completed under the guidance of **Manan Rasool Sir**, helping me understand how raw data is transformed into meaningful insights.

🔍 What I implemented:
• Handling missing data and removing less useful features (e.g., Cabin column)
• Extracting meaningful information from raw data (like deriving Titles from names)
• Applying transformations and mapping categorical values
• Using functions and regular expressions for data extraction
• Inspecting and validating data using methods like `.head()`

📊 Key Learning Outcomes:
• Importance of data preprocessing before analysis
• How feature engineering improves dataset quality
• Practical use of Python libraries for real-world datasets
• Understanding structure and patterns in raw data

🛠️ Tools & Technologies:
Python | Jupyter Notebook | Pandas | Regular Expressions

💡 This project strengthened my foundation in **data preprocessing and feature engineering**, which are crucial steps in any data analysis or machine learning workflow.
